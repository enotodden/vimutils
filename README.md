vimutils
========

Vim utilities


+ *ve* - Open files as buffers in current vim server

        $ ve myfile.txt myotherfile.txt 

+ *vt* - Open files as tabs in current vim server
        
        $ vt myfile.txt myotherfile.txt

+ *vls* - List running servers
    
        $ vls

+ *vbls* - List open buffers

        $ vbls

+ *vdb* - Dump buffer N

        $ vdb 19
